# UART Loopback Verilog Project

## Description
This project implements UART communication using Verilog HDL.

Features:
- UART Transmitter
- UART Receiver
- Baud Rate Generator
- Internal Loopback
- Testbench Simulation

## Baud Rate
9600 baud

## Clock Frequency
100 MHz

## Tools Used
- Vivado
- Verilog HDL

## Simulation Result
Successfully transmitted and received:
41h
